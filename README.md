## Install this app

1. Clone this repo

```sh
git clone https://github.com/kingkinfajarr/image-gallery-react.git
```

2. Install dependencies

```sh
npm install
```

3. Chance .env.example to .env and chance with your [Pixabay API](https://pixabay.com/api/docs/) key
4. Start app

```sh
npm start
```
